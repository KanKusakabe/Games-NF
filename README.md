# Games-NF

ゲームの**実人間プレイ**を条件付き **Normalizing Flow** で学習し、潜在に
**「右＝上手・左＝下手」の技量軸**を通す試み。1人用/協力/対戦の複数ゲームを**同じ基底**で扱う。

**▶ 結果ページ（GitHub Pages）**: https://kankusakabe.github.io/Games-NF/

## Phase 0（基底の決定）— 完了
Ms.パックマンの実人間プレイ（Atari Grand Challenge・382エピソード、スコア10〜18,981）で、
**プレイスタイルだけ**（行動方向分布＋切替率＋エントロピー・生存/スコアはリークさせない）から技量軸を学習。
- **教師つき技量軸（z₁≈標準化スコア）**：初心者vs熟練 **AUROC 0.80**／z₁と実スコア **r=0.50**／held-out NLL −0.41
- 教師なし＋後付け方向：AUROC 0.70
- → **基底＝ガウス＋直線の技量軸**を採用（技量は単調・順序なので円は使わない。円は移動方向用）。

## 設計方針（全ゲーム共通）
- 基底：**ガウス**、技量は**1本の直線軸 z₁**（スコア/レートで整列）。多次元技量は z₂ 以降。
- **ゲームIDを条件にした単一フロー**で全ゲームを横断（技量軸・方向の意味を共通化）。
- 技量ラベル：1人用=スコア、協力=チームスコア、対戦=レート/段級位（各ゲーム内で標準化）。

## 予定ラインナップ
