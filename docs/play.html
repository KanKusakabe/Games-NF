触って体験：技量軸で「採点」と「一歩上の助言」

学習した技量軸 z₁ を実際に動かして体験する。モデル推論なし・100%実データ由来——あなたの打ち方に最も近い実在プレイヤーを探して採点する。

使い方：ゲームを選び、スライダーで打ち方（プレイスタイル）を動かす。その場で ①採点ゲージ（近い実プレイヤーの技量パーセンタイル）②助言チップ（上位層に届いていない項目だけ ▲増やす／▼減らす） ③近い実プレイヤー例（実際のスコア/レート等）が更新される。「実プレイヤー読込」でスライダーを実データの1人にセットも可能。 さらに全6ゲームとも下部で実際に遊んで採点できる（プレイ中はライブ・コーチ＋参考手/経路ヒント付き）。

採点＝標準化特徴空間での kNN(k=25) 近傍の z₁ パーセンタイル平均。助言＝各レバーで「実技量 上位1/3 の平均（標準化）」に届かない向き（phase2.py と同じ tercile 方向）。データは python -m gamesnf.export_app で再生成。
