やり方（全ゲーム共通）：各プレイからスタイル特徴（行動方向の分布＋切替率＋エントロピー、 生存/スコアはリークさせない）を作り、基底＝ガウス＋直線の技量軸の条件付きNSFを学習。 学習時に z₁ ≈ 標準化スコア を課し、z₁ そのものを技量軸にする（Phase 0でMs.パックマンにて 教師つき軸 AUROC 0.80 > 後付け 0.70 と確認して基底を決定）。技量は単調・順序尺度なので直線で、 円環は移動方向用に温存。
